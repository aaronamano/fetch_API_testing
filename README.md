## Overview
this a simple repo on how to call a built in API called [PokeAPI](https://pokeapi.co/) using the **fetch chaining method** and **async/await function**

## Project
this sample mini project was inspired by the youtube video [How to FETCH data from an API using JavaScript ↩️](https://youtu.be/37vxWr0WgQk?si=zxeBSLtA28ujvq-D) by Bro Code. you input a pokemon name and then its image sprite is returned.

## Test
you can run `node fetch.js`, `node fetch2.js` and/or `node async.js` to view the responses in your terminal.
